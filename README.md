## Video Streaming Platform Built with ReactJs, Redux, Redux-Toolkit ,React-Router-Dom and TailwindCss

## Features:
### Highly performant search

### Search with debouncing 

### n-level nested comments

### Caching of search results for optimal performance

### Used YouTube Live API's to generate videos 

### AutoPlay on Click 

### Live Demo: (https://my-fav-youtube.vercel.app/)
### Source Code: (https://github.com/anshu292/my-fav-youtube)

Note- Use Allow Cors Extension for Search Suggestions on your UI and project currently supports desktop version.
